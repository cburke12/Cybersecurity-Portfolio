# Module 6 - Tcpdump & Wireshark

## Objective
Capture and analyze network packets using Tcpdump and Wireshark to identify suspicious traffic patterns, protocol anomalies, and potential threats in real network data.

## Skills Demonstrated
- Packet Capture (Tcpdump, Wireshark)
- Protocol Analysis (TCP, UDP, HTTP, DNS, ICMP)
- Traffic Filtering & Display Filters
- Network Forensics
- IOC Identification from Packet Data
- PCAP File Analysis

## Tools Used
- Wireshark
- Tcpdump
- Linux CLI
- Sample PCAP files
