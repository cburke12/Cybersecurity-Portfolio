# Module 4 - Active Directory & IAM

## Objective
Deploy and configure a real Active Directory environment using Windows Server 2025 and VirtualBox. Provision users at scale using PowerShell, manage group policies, and demonstrate Identity and Access Management (IAM) principles in a hands-on home lab.

> **Note:** This is my personal spin on the standard portfolio. Instead of just reading about Active Directory, I built the whole thing from scratch in my home lab.

## Skills Demonstrated
- Active Directory Domain Services (AD DS)
- Windows Server 2025 Administration
- PowerShell Scripting (bulk user provisioning)
- Organizational Units (OUs) & Group Policy Objects (GPOs)
- Domain Controller Setup & Configuration
- Client VM Domain Join
- Identity and Access Management (IAM)
- User / Group / Role Management
- Authentication & Authorization
- Cryptography Basics
- Asset Classification
- Threat Modelling Analysis

## Tools Used
- Windows Server 2025
- Windows 11 (client VM)
- VirtualBox
- PowerShell
- Active Directory Users and Computers (ADUC)
- Group Policy Management Console (GPMC)

## Lab Setup
- Domain Controller: Windows Server 2025 in VirtualBox
- Client Machine: Windows 11 VM joined to domain
- 1,000+ users provisioned via PowerShell automation script
- OUs structured by department (IT, HR, Finance, Engineering)
- GPOs applied for password policy, desktop lockdown, software restrictions
