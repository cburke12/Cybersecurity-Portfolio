# Module 1 - Security Audit

## Objective
Conduct a mock internal security audit for a fictional organization using the NIST Cybersecurity Framework (CSF) and NIST Risk Management Framework (RMF). Identify gaps in controls, assess compliance posture, and produce a findings report with recommendations.

## Skills Demonstrated
- NIST CSF (Identify, Protect, Detect, Respond, Recover)
- NIST Risk Management Framework (RMF)
- CIA Triad (Confidentiality, Integrity, Availability)
- Security Controls Assessment
- Risk Scoring & Prioritization
- Compliance Checklist Development
- Incident Response Playbook Review

## Tools Used
- NIST CSF Reference Spreadsheet
- Google Docs / Markdown for documentation
