# Module 5 - Detection & Incident Response

## Objective
Follow the full incident response lifecycle for simulated security events. Document incidents using an incident journal, practice escalation procedures, and apply NIST SP 800-61 guidelines.

## Skills Demonstrated
- Incident Lifecycle (Preparation, Detection, Containment, Eradication, Recovery)
- Incident Journal Documentation
- NIST SP 800-61 Incident Response Framework
- Alert Triage & Escalation Procedures
- Post-Incident Analysis & Lessons Learned
- IOC (Indicator of Compromise) Identification

## Tools Used
- Incident Journal (Google Docs / Markdown)
- Splunk (log-based detection)
- Suricata (alert generation)
- Linux CLI
