# Module 3 - Linux & SQL

## Objective
Use the Linux command line and SQL to perform security tasks: managing file permissions, analyzing logs, filtering database queries to investigate suspicious activity, and hardening system access.

## Skills Demonstrated
- Linux Command-Line Interface (CLI)
- File System Permissions (chmod, chown)
- User & Group Management
- Bash Scripting
- SQL SELECT, WHERE, JOIN, LIKE queries
- Log File Analysis
- Database Security Filtering

## Tools Used
- Linux (Ubuntu)
- Bash / Terminal
- SQLite / MySQL
- grep, awk, cut, find

- 3.2 - Apply Filters to SQL Queries *(in progress)*
