# Module 8 - Python Security Automation

## Objective
Write Python scripts to automate common security tasks: log parsing, file integrity checking, alert generation, and security tool integration. Apply PEP 8 standards and write clean, documented code.

## Skills Demonstrated
- Python Programming (PEP 8)
- Log File Parsing & Analysis
- File I/O Operations
- String Manipulation & Regular Expressions
- Security Automation Scripting
- Algorithm Design (hash tables, sliding windows)
- Data Structure Selection for Security Use Cases

## Tools Used
- Python 3
- VS Code / Cursor
- Linux CLI
- GitHub

- 8.2 - File Integrity Checker with Hash Comparison *(in progress)*
